Visão Geral

Este projeto consiste na implementação do popular jogo "Snake" rodando em um dispositivo ESP32. O jogo se comunica com uma página HTML para atualização de pontos e armazena os resultados na plataforma ThingSpeak para futuras consultas. O objetivo foi integrar hardware e software para criar uma experiência interativa e conectada.

Desenvolvimento do Jogo

  • Implementação do jogo Snake no ESP32
  • Controle de pontuação e lógica do jogo
  • Exibição de pontuação no display do ESP32

Integração com ThingSpeak

  • Envio de pontuações para o ThingSpeak via "myWriteAPIKey"
  • Leitura do maior recorde via "myReadAPIKey"
  • Armazenamento de até 12 registros no ranking

Dashboard HTML

  • Criação de uma página HTML+JavaScript para exibir o ranking
  • Destacar o maior registro no ranking
  • Integração com o ThingSpeak para exibição dinâmica
