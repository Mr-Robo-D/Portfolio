Visão Geral

Este relatório descreve a migração da infraestrutura da Abstergo Industries para a nuvem da AWS. O objetivo foi modernizar operações e reduzir custos, substituindo hardware local por serviços gerenciados como S3 para armazenamento, RDS para banco de dados e EC2 para aplicações...

Diagrama da Arquitetura

Tecnologias Utilizadas

Amazon S3
Amazon RDS
Amazon EC2

Arquitetura e Padrões

  • Armazenamento de objetos com Amazon S3.
  • Banco de dados relacional gerenciado com Amazon RDS.
  • Camada de computação com Amazon EC2.
  • Diagrama representando a topologia e o fluxo entre os serviços.

Requisitos do Sistema

  • Conta AWS para replicação/estudo da arquitetura.
