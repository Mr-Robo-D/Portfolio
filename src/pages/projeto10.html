Visão Geral

Conjunto de desafios de programação em Java desenvolvidos durante o bootcamp GFT. Abrange problemas de lógica, estruturas de dados e operações de entrada/saída.

Exemplos/Ilustrações

Tecnologias Utilizadas

Java

Descrição Técnica

Resolução de problemas algorítmicos em Java, com organização por módulos/pacotes e foco em clareza, legibilidade e testes locais. As soluções adotam boas práticas de entrada/saída e tratamento de erros.

Principais Funcionalidades

  • Implementações de algoritmos clássicos e intermediários.
  • Estruturas de dados básicas (arrays, listas) e manipulação de strings/números.
  • Leitura e escrita de dados via console.

Arquitetura

Organização por pacotes para separar tópicos/temas dos desafios. Código simples e direto ao ponto para facilitar o estudo e a revisão.

Requisitos do Sistema

  • JDK instalado (versão compatível com o projeto).
  • Ferramenta de build Java (Maven ou Gradle) opcional para execução/organização.
