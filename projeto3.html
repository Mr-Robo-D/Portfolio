Visão Geral

Este projeto consiste em uma aplicação básica de CRUD (Create, Read, Update, Delete) desenvolvida em Java utilizando Spring Boot e MongoDB. O objetivo foi criar uma API RESTful funcional para gerenciar dados de usuários, explorando as funcionalidades do Spring Data MongoDB.
